# Sistema de Gestión de Asistencias y documentación - API 🚀

Este es el núcleo del sistema desarrollado durante las prácticas en T3M E.I.R.L. Gestiona la lógica de negocio, autenticación y persistencia de datos.

### 🛠 Tecnologías
- **Framework:** Laravel 10.x
- **Ambiente:** Docker / Laravel Herd
- **Base de Datos:** MySQL

### 📦 Instalación
1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Copiar `.env.example` a `.env` y configurar la base de datos.
4. Ejecutar `php artisan migrate`.
